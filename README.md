# Pamela Austin - Portfolio Website

A modern, responsive portfolio website showcasing data engineering, analytics, and business intelligence case studies.

## Features

- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Five Core Expertise Areas**:
  1. Business Intelligence (BI) Analyst / Developer
  2. Healthcare Data Analyst
  3. Marketing Business Analyst
  4. Financial Analyst (Data-focused)
  5. Product Analyst

## Structure

```
/
├── index.html          # Homepage
├── archives.html       # Archive of projects
├── resume.html         # Resume page
├── case-study-*.html   # Individual case studies
├── styles.css          # Shared styling
├── script.js           # Shared JS (theme, sidebar/mobile nav, effects)
├── assets/
│   ├── docs/
│   └── images/
└── README.md
```

## Customization

### Update Contact Information

Edit the following in `index.html` and other site pages as needed:
- Email address
- LinkedIn URL


## Deployment

### Option 1: GitHub Pages
1. Create a GitHub repository
2. Push these files to the repository
3. Go to Settings > Pages
4. Select main branch as source
5. Your site will be available at `https://yourusername.github.io/repository-name`

### Option 2: Netlify
1. Create a Netlify account
2. Drag and drop the Website folder
3. Your site will be instantly deployed

### Option 3: Custom Domain
Upload files to your web hosting via FTP/SFTP.
