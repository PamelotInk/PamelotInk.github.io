# Resume PDF Instructions

## To create a professional PDF resume:

### Option 1: Use Microsoft Word or Google Docs
1. Create your resume in Word/Google Docs
2. Export as PDF (File → Save As → PDF or File → Download → PDF)
3. Save as `Pamela_Austin_Resume.pdf` in this folder

### Option 2: Use the browser print (with proper settings)
1. Open resume.html in Chrome
2. Press Ctrl+P (or Cmd+P on Mac)
3. Change "Destination" to "Save as PDF"
4. Click "More settings"
5. **UNCHECK** "Headers and footers"
6. Set margins to "Default" or "Minimum"
7. Save as `Pamela_Austin_Resume.pdf` in this folder

### Option 3: Use a free online tool
1. Use https://www.ilovepdf.com/html-to-pdf
2. Or use Canva to design a professional resume
3. Export as PDF and save here
